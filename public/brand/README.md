# Brand logo assets (§02 Logo System)

Place **unmodified** official files from the brand package here:

| File | Use |
|------|-----|
| `logo-lockup-light.png` or `.svg` | Full lockup on mint / white backgrounds |
| `logo-lockup-dark.png` or `.svg` | Full lockup on navy / dark backgrounds |

The site uses `/icon-247.png` as the **icon mark** (favicon, compact UI) until lockups are added. The `Logo` component falls back to the icon mark if lockup files are missing.

**Do not:** recolor, add shadows/glows, stretch, rotate, or recreate the lockup in CSS/type.
